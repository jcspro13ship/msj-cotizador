# MSJ — Cotizador de Precios

Aplicación web para cotización con tres listas de precios: Punto de Venta, Veterinario y Público.

## Configurar Google Sheets

1. Abre tu Google Sheets → **Archivo → Compartir → Compartir con otros**
   - Cambia a "Cualquier persona con el enlace" → Ver → Listo
2. Copia el **ID** de la URL de tu hoja
3. En la app haz clic en ⚙️ **Sheets** y pega el ID y el nombre de la pestaña

### Columnas requeridas en fila 1:
```
Codigo | Producto | Categoria | Precio PV | Precio Vet | Precio Pub
```

## Publicar en GitHub Pages

1. Sube estos archivos a un repositorio en GitHub
2. Ve a **Settings → Pages → Source: main branch / root**
3. Tu app estará en `https://TU_USUARIO.github.io/NOMBRE_REPO`
